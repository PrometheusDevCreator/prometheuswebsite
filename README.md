# ELAN Training Solutions

A sophisticated animated landing page for **Prometheus Education Systems** - an educational technology platform specialising in AI-powered training course development.

## Live Demo

[View Live Site](https://prometheuseducationsystems.com) *(coming soon)*

## Overview

ELAN (Educate, Learn, Advance, Navigate) Training Solutions provides comprehensive client-facing training systems powered by the Prometheus suite of educational tools.

### Features

- **3-page animated experience** with smooth transitions
- **Page 1**: Animated landing with login flow
- **Page 2**: Company information and product overview
- **Page 3**: Product showcase grid featuring the Prometheus suite
- **Contact overlay**: Click the elephant logo for contact details
- **Responsive design**: Works on desktop, tablet, and mobile
- **Self-contained**: All assets (images, audio) embedded as base64

### Products Showcased

| Product | Description |
|---------|-------------|
| **Prometheus Core** | Comprehensive Training Course Development System |
| **Prometheus Knowledge Engine** | Proprietary Information Resource Governance AI |
| **Formatting Tools** | Document Conversion, Calibration & Template Mapping |
| **Standalone Learning Apps** | Bespoke Subject-Based Individual Self Learning |
| **Language Learning** | Customised Language Tutors *(Coming Soon)* |
| **Technical Documentation Aids** | Enhanced Technical Manual Navigation Tools |

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts: Cinzel, Rajdhani, Cormorant Garamond
- CSS animations and transitions
- Web Audio API for click sounds
- Responsive CSS Grid and Flexbox layout
